SAR Interferometric Analysis

1. Objetivo

El objetivo es mapear huellas de zonas urbanas mediante la interferometría SAR. Se hará uso tanto de la amplitud como la fase de la señal de dos imágenes Sentinel 1A dando como resultado tres productos diferentes que se usaran para filtrar zonas edificadas. Los dos primeros productos son la amplitud calibrada para cada imagen y el último es el de la fase que surge de la coherencia interferométrica de las dos. La coherencia de cada pixel se compone de distintos factores donde el factor geométrico y el temporal son los que más influyen sobre este. El primer factor puede ser eliminado pero el factor temporal depende estrictamente de las características de la zona de estudio. Esto permite encontrar cambios entre diferentes espacios temporales y en este caso especial como las zonas urbanas se componen de estructuras inamovibles que se mantienen en el tiempo la coherencia calculada correspondería ser alta.

2. Caso de estudio

El lugar de estudio es el rio Nilo antes de arribar al Cairo y Alejandría.

Ilustración 1: Zona de estudio remarcada en rojo.

El río Nilo tiene una longitud aproximada de 6853 km y es el más largo del mundo. Nace en el lago Victoria en Tanzania y atraviesa 10 países africanos hasta desembocar en el Mar Mediterráneo, formando el delta del Nilo sobre el cual se sitúan El Cairo y Alejandría. Rodeado por el desierto más grande del planeta, el Sahara, obligó históricamente a los Norteafricanos a fértiles viajes al Mediterráneo y al río Nilo. Funciono como fuente inagotable de recursos y como principal vía de comunicación hacia el Mediterráneo. De todo el trayecto del rio se eligió un tramo dentro de Egipto para buscar huellas de ciudades que se asentaron a las orillas del mismo (Ilustración 1).

3. Datos

Para llevar a cabo el trabajo se usaron dos imágenes SAR descargadas del hub de ESA-Copernicus con un nivel de procesamiento SLC. Ambas de la misión satelital Sentinel 1A banda C con modo de adquisición TOPSAR - IW y una distancia temporal de dos meses sobre el mismo sector. El método IW adquiere datos con una franja de 250 km a una resolución espacial de 5 m x 20 m (SLC).

4. Metodología

El procesamiento de las imágenes se llevó a cabo en el software SNAP siguiendo los siguientes pasos.

  1. Se realizó un split para la polarización VV de modo de obtener solo la rodaja de la imagen que cae sobre el rio.
  2. Incorporación de los archivos de orbita precisa que se descargaron automáticamente desde el software para mejorar las correcciones geométricas sobre el producto final.
  3. Eliminación del ruido térmico termal S1.
  4. Calibración con banda sigma 0 de la imagen para convertir el pixel en una cantidad física, en este caso la retrodispersión.
  5. Se unió zonas de demarcación de la imagen con un deburst.
  6. Aplicación de multilooking. Se pasa el pixel de un rectángulo a cuadrado (13.98 x 13.98 m)
  7. Filtrado Speckle a cuesta de perder resolución.
  8. Corrección por terreno rango-doppler. Se corrige con un MDE de 90 m descargado automáticamente del software y se georreferencia en el sistema WGS84.

El proceso fue similar para las dos imágenes descargadas obteniendo dos productos para las mismas coordenadas geodésicas en dos espacios temporales distintos. Asimismo, a los productos obtenidos del paso 2 (luego de la aplicación de orbita precisa) se los uso para calcular la interferometría de coherencia. Para comenzar se remuestrea (coincidir pixeles) de una imagen sobre la otra mediante una corregistración y se calcula la coherencia. Al resultado obtenido se le aplica el mismo procesamiento que se le aplico a los anteriores.

De esta forma, se obtiene tres productos calibrados sobre la misma zona de estudio. Se los acopla sobre un mismo producto con stack para usarlos como tres bandas distintas. Con los productos de amplitud calibrados se calculó la media y la diferencia entre los mismos.

Tabla 2.

La representación del faso RGB se hace con la composición de la Tabla 2. Por último, se aplica una máscara de manera tal de destacar las zonas urbanas.

if valores_coherencia > 0.5 and valores_coherencia < 0.7 and media_dB > -8 and diferencia_dB > -5 then 1 else 0.

Básicamente, se filtra de manera condicional los valores de pixel de la imagen de coherencia entre 0.5 y 0.7 y los valores de media y diferencia mayores a -8 y -5 respectivamente. Si se cumple, el valor de la máscara vale 1 caso contrario 0.

Se trabajo con decibeles (dB) para convertir las bandas de escala lineal a logarítmicas. Permite manipular el histograma para mejorar la visualización de los productos. Finalmente, la salida gráfica se diseño en el software QGIS.

5. Resultados

El primer resutlado obtenido es el mapa de coherencia y su histograma:

Ilustaración 2: Mapa de coherencia. Ilustaración 3: Histograma de coherencia.

La coherencia describe la similitud de señal reflejada del radar entre dos imágenes, una es la master y la otra la “slave”. El rango de la coherencia va de 0 a 1, los valores más cercanos a uno (zona blanca) quiere decir que existe una coherencia entre las dos imágenes y en la zona oscura lo contrario. Se puede ver que en las zonas de vegetación alrededor del rio existe poca coherencia entre las dos. Sin embargo, dentro de estas si se puede observar sectores brillantes que dan un indicio a zonas edificadas y en las zonas de color blanco opaco el desierto que no varia con en el tiempo.

Ilustración 4: Imagen satelital Bing y producto obtenido en falso RGB en escala 1.500.000. Se destaca zona de vegetación alrededor del rio.

La imagen en falso RGB proporciona una idea del uso del suelo. En rojo y naranja los pixeles son de baja retrodispersión y alta coherencia indicando zonas de agricultura o zonas de suelo desnudo, en este caso el desierto. En cambio, en color amarillo brillante resultan pixeles de alta retrodispersión y alta coherencia dando como resultado zonas edificadas. Además, se aprecia en azul oscuro cuerpos de agua y en verde extensos campos de vegetación.

Finlamente el producto final:

Ilustración 5: Producto final con un mapa base de OM. Se resalta huellas urbanas en color verde.

Aplicando el filtro explicado anteriormente, se arriba al producto final con las huellas urbanas en representadas en color verde para que se logren visualizar en la imagen.

6. Discusión

Para llegar al objetivo del trabajo se utilizó un filtro con los tres productos obtenidos. El problema es que se tuvo que definir el filtro final de manera manual, a prueba y error, observando los valores de decibeles (dB) de los pixeles hasta llegar al producto esperado. En una primera instancia se obtuvo un producto totalmente diferente, es decir, se destacaban zonas no edificadas en conjunto con edificadas. Para esto se propone buscar alguna técnica especial que directamente logre describir no de manera manual sino con un contexto estadístico. Asimismo, para mejorar el análisis se puede agregar otro mapa de coherencia a partir de otras dos imágenes en otro tiempo para comparar cambios entre ambos.

7. Conclusiones

El desarrollo del trabajo logro describir el objetivo propuesto. Se pudieron observar los beneficios del uso de imágenes satelitales SAR para la distinción de objetos en el terreno. Se logró diferenciar entre zonas desérticas y campos extensos de vegetación. Los últimos, se encontraban alrededor de cuerpos de aguas como era de esperar. Como consecuencia, la mayoría de las ciudades edificadas que se alcanzaron a filtrar están el rededor de estos. Por último, se practicó el procesamiento de imágenes SAR, así como la aplicación de una de las técnicas de la interferometría SAR.

SAR Image Preprocessing

El objetivo es llevar a cabo la descarga, exploración y preprocesamiento de dos imágenes SAR Sentinel-1, generando información geoespacial para visualizar y analizar, en un Sistema de Información Geográfica (SIG). Como caso de estudio se eligió la erupción volcánica de La Palma de 2021. Se inició a las 14:10 horas (UTC) del 19 de septiembre en el paraje de Cabeza de Vaca, cercano a la localidad de El Paraíso del municipio de El Paso, en la isla española de La Palma. El mismo continua en actividad hasta entonces.

Se seleccionaron tres imágenes SAR del satélite Sentinel 1B con modo de adquisición TOPSAR - IW con un nivel de procesamiento GRD. La imagen 1 es un año antes del inicio de la erupción (2020/09/21) y la imagen 2 (2021/09/20) y 3 (2021/10/02) están separadas por 12 días. Las imágenes fueron descargadas de Copernicus Open Access Hub. La primera imagen es de pasada ascendente y el resto son descedente.

Ilustración 1: Productos posprocesamiento en SNAP. Zona de estudio: latitud 28°37 y longitud 17°52.

Se uso la polarización VV para analizar el efecto de la lava sobre la vegetación. Además, el falso RGB se compone por la polarización VV en rojo, HV en verde y la resta de ambos para el azul. Se convirtió a decibeles para poder manipular el histograma y a su vez mejorar la visualización de los productos finales. Se debe tener en cuenta que la imagen 1 es ascendente y el resto son tomadas de manera descendente. Esto puede provocar algunas diferencias entre la comparación de las imágenes.

De la ilustración 1 en un principio se observa baja retrodispersión (baja intensidad dB) para las tres imágenes en la zona de la Montaña Quemada. Ahí se encuentra la colada lávica del Volcán Tacande que da a un sector de poca vegetación. En cambio, donde la retrodispersión es alta (amarillo brillante en RGB) se encuentran zonas urbanas. Asimismo, tanto para los productos de la fecha 2021/09/21 y 2021/09/20 no se observa algún cambio alguno en el entorno del volcán Tajoigate pero si se aprecia unos días después (2021/10/02) donde el color rojo en el falso RGB cobra mayor intensidad.

Este trabajo fue parte del curso de complementación “Radar de Apertura Sintética: fundamentos y ténicas interferométricas” de la Universidad de Buenos Aires.

La Copa America 2021 de Brasil en números

Brasil hace 18 partidos que no pierde y la última vez que lo hizo en su propio país fue ante Países Bajos por el tercer puesto del Mundial del 2014. Desde entonces no conoce la derrota el ahora actual equipo de Tite.

Sus números en la actual Copa América 2021 reflejan un equipo superior a sus rivales en la mayoría de los aspectos. No obstante, el rendimiento del equipo fue de mayor a menor.

En total acumuló 12 goles con una expectativa de gol de 12.4. Es decir, que genera situaciones de calidad y a su vez es efectivo frente al arco. Los goleadores del equipo son Paquetá y Neymar con dos goles ambos.

Desde el aspecto defensivo prioriza defenderse con la pelota. Recibió 39 disparos donde 16 fueron al arco. En conjunto, esos 16 disparos generaron una expectativa de gol de 3.3. Frente a Chile fue la única vez que se vio superado en la posesión de pelota y fue un caso excepcional porque jugo todo el segundo tiempo con un jugador menos.

Analizando la plantilla de Brasil, Tite le dio minutos a la mayoría de los jugadores de la lista. En los últimos tres partidos el esquema inicial fue “4-2-3-1” luego de haber utilizado el “4-4-2” y “4-3-3”.

En función a los minutos jugados se puede dar por hecho que el lateral derecho (Danilo), los centrales (Thiago Silva y Marquinhos), el doble 5 (Casemiro y Fred) y Neymar con Richarlison son la columna vertebral del equipo. El lateral izquierdo se lo turnaron entre Lodi y Alex Sandro, donde parece que Lodi gana la pulseada por haber jugado los últimos dos partidos de eliminación directa. A los últimos 3 jugadores restantes Tite suele elegirlos en función del rival de turno. Los minutos entre estos delanteros se encuentran distribuidos homogéneamente.

En los siguientes gráficos los valores están ajustados por 90 minutos (tiempo jugado en cancha) por lo que jugadores con pocos minutos pueden tener números alterados. Dos ejemplos de esto son Vinicius Junior y Douglas Luiz que pueden tener números por encima del resto por haber jugado menos de 90 minutos. Esto no invalida sus acciones dentro del campo, solo que al ajustarse por minutos jugados los sobre exagera, a la vez que comparar los eventos entre los distintos jugadores del plantel. El fin es analizar ciertas métricas de los jugadores que pueden caracterizar el estilo de juego de Brasil y su funcionamiento, así como también sus jugadores destacados.

Siguiendo adelante, Brasil se caracteriza por ser un equipo agresivo tras perdida y sus presiones en el último tercio lo reflejan. Es el segundo equipo que más presiona en el último tercio respecto a la media detrás de la selección de Ecuador.

Hilando fino entre los jugadores, Fred es el segundo jugador que más presiona de todo el plantel por debajo de Vinicius Junior. Del top 10 de los jugadores que más presionan, solamente 3 no son delanteros, es más, los que suelen jugar por las bandas son los que más exigen al rival a elegir el pase de manera inmediata.

En lo que respecta a la recuperación de la pelota, Casemiro es el que manda y lo sigue Fabinho, que suele ser su reemplazante. A estos los acompaña Fred que es el motorcito de la mitad de la cancha.

En la progresión de pases y pases al último tercio nuevamente sobresale el doble 5 de Brasil que es desde el cual el equipo empieza a crecer. También se hacen notar los números de Danilo y Alex Sandro, ambos laterales, que suelen ser opción de pase cuando el equipo sale por abajo.

Por otro lado, lo que no aporta Neymar en la recuperación de la pelota lo compensa en la generación de juego. Es el primero en las métricas “Key Passes” o pases claves, expectativa de asistencia y gol dentro del equipo. Asimismo, hasta la fecha es el mayor asistidor con 3 asistencias y el segundo de toda la Copa por detrás de Lionel Messi (5 asistencias).

Lo mismo ocurre con los toques dentro del área penal. Sin sorpresa alguna, Ney y Richarlison se encuentran entre los jugadores que más tocan el balón en el área rival. Un detalle no menor es la influencia de los toques en el área de los laterales, así como también su aporte en la métrica SCA en sus siglas en inglés. Shot-Creation Actions se define como las dos acciones ofensivas que directamente derivan en un disparo, como pueden ser pases o dribbles.

En conclusión, me quedo con el doble 5 de Brasil, y sobre todo la gran copa que está haciendo Fred. Es uno de los jugadores más influyentes del equipo con Neymar y cumple un rol fundamental dentro del sistema de juego.

Brasil es un equipo donde todos presionan salvo Neymar que se guarda para hacer daño en último tercio. Es un equipo al que le patean poco y convierte todo lo que genera en el arco contrario.

Todos los datos fueron obtenidos de Statsbomb via Fbref. Algunos jugadores se identifican en otra posición por la manera en que los clasifico Fbref.

Players Scouting Tool

Scouting Tool is a web application that lets you search, find, and analyze football players within the Top 5 European Leagues. The application is built with python library Streamlit. The open data is scraped from Statsbomb via FBref and Transfermarkt for seasons 2017-18 to 2020-21. Only field players with more than 900 minutes in each season were considered. Event values are adjusted per 90 minutes and for the moment defensive metrics are not adjusted by squad possession.

The objective of the app is to filter and display football (soccer) event data in an elegant and efficient way using visualizations. The app is divided in four components:

  • Advanced search

  • Compare

  • Player Profile

  • Scatter Plot

Advanced search

This component allows you to create a shortlist from a pool of players. The pool can be filtered by personal information and event metrics. If you are curious to find similar players, you have the option to do it. The similarity model is built with PCA variable reduction and euclidean distance. It is a basic and simple model which might have it flaws but it does the work.

Compare

The next component allows to compare players based on a template for a selected position. Each template contains the events that are considered essential for each position. If you do not want to compare against a player, you can do it by the mean of all events of all players in a selected league, or a specific position. The final output is a radar plot and a table. The radar boundaries represent the maximum values of production by all the pool of players in each metric.

Plaer Profile

Player profile component shows a bar plot for a specific player. The viz displays percentiles ranks within player position for chosen league or TOP 5 European Leagues. It also incorporates the absolute value of each event. It is a neat way to see player performance in a season compared to the rest.

Scatter Plot

A simple plotly scatter plot. It is built to analyze the distribution between two variables and find outliers. You can filter the data with parameters and the plot is interactive. It enables you pan, zoom in and out, and square or lasso select. Another cool option is that you can display a hover if you put the mouse on top of each dot.
